# cloud_music
造轮子复刻网易云音乐, 网易无版权的歌曲自动切换至QQ曲库

# Build
    # 安装
    git clone https://github.com/EnzoVVV/cloud_music.git
    npm run init
    
    # 运行
    npm start
    
    # chrome下按 F12, Ctrl + Shift + M 进入移动端模拟模式
